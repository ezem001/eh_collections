# Stage workspace

Підключіть саме цю папку до Postman workspace `Stage`.

Очікувана структура після підключення і першого pull:

- `.postman/resources.yaml`
- `postman/collections/...`
- `postman/environments/...`
- `postman/globals/...`
- `postman/flows/...`
- `postman/sdks/...`
- `postman/specs/...`
